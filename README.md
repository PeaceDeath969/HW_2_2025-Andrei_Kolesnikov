# Text Analysis & Sentiment Classification

## Описание
Этот проект включает в себя два основных этапа:
1. **Анализ текстов песен** — применяются методы векторизации текста (TF-IDF, CountVectorizer, Word2Vec).
2. **Классификация отзывов** с использованием модели BERT для анализа тональности (позитивный/негативный).
